Data Handling 3 - Glossary

Glossary DH3
Binary Mask: A mask that isolates pixels whose intensity value is below a certain threshold. Everything below this threshold can be assigned to, for instance, a value of 1 (representing True), and everything above will be assigned a value of 0 (representing False).
Channel: A single component of a colour image, such as the Red, Green, or Blue layer in an RGB image.
Colour Map: A method to map numerical data to colours, allowing single-channel images to be visualized with false colours (e.g., 'viridis' or 'magma').
Coronal Plane: A viewing plane in 3-D imaging that divides the body into front and back sections.
Downsampling: The process of reducing the number of pixels in an image (resolution) to decrease file size or memory usage.
False Colour: Colours assigned to intensity values in a greyscale image to enhance visualization, not representing the true colour of the subject.
Greyscale: An image format where each pixel represents a single intensity value, usually ranging from 0 (black) to 255 (white).
Histogram: A graphical representation of the distribution of pixel intensity values in an image.
Mask: A filter applied to an image to isolate specific regions based on pixel intensity or other criteria.
MRI: Magnetic Resonance Imaging. A medical imaging technique that frequently produces 3-D image data.
NIfTI: Neuroimaging Informatics Technology Initiative (NIfTI) open file format. A common file format for neuroimaging data (e.g., MRI scans).
Patching: A technique to divide a large picture up into smaller chunks that are termed patches, often used to handle large images that are difficult to process in memory.
Pixel: In digital imaging, a pixel is simply a numerical value that represents either intensity, in a greyscale image, or colour intensity, in a colour image. The name 'pixel' is an amalgamation of 'picture' and 'element'.
Resolution: The total number of pixels in the image.
RGB: A colour model where any colour is a composite of the intensity value for Red, Green, and Blue channels.
Sagittal Plane: A viewing plane in 3-D imaging that divides the body into left and right sections.
Slice: A two-dimensional cross-section extracted from a three-dimensional image volume.
Thresholding: A method of image segmentation that converts a greyscale image into a binary mask based on whether pixel intensities are above or below a specified value.
Transverse Plane: A viewing plane in 3-D imaging that divides the body into superior (top) and inferior (bottom) sections.
Voxel: Volume pixel. A value on a regular grid in three-dimensional space.
